# Photographer Portfolio (Presentation Style)

添付のプレゼン資料デザインをベースに、左サイドバーとカード型セクションで構成されたフォトグラファーのワンページポートフォリオです。黄色とライトグレーを基調に、ホーム／自己紹介／レジュメ／ポートフォリオ／推薦文／お問い合わせまでをスクロールで閲覧できます。

## 構成
- `index.html` — 参照デザインを再現したHTML。左右サイドバーと6つの主要セクションを配置。
- `styles.css` — 新しいライトトーンのデザインシステム、レイアウト、レスポンシブ調整を収録（既存のスタイルからの上書き）。
- `app.js` — スクロール位置に応じたナビゲーションのアクティブ制御とスムーススクロール処理。
- `concept/` — プロフィールで使用している参考写真。

## セクション概要
- **Home**: 自己紹介ヒーロー、統計値、丸型ポートレート＆アイコン。
- **About**: 「What I do」と「Fun Facts」をカード表示。
- **Resume**: Education / Experience をタイムライン形式で表示。
- **Portfolio**: Print / Web / Photography のギャラリーとフィルターチップ（UI見本）。
- **Testimonials**: 3件の推薦コメントカード。
- **Contact**: 連絡先と問い合わせフォーム。

## 使い方
1. リポジトリを取得して任意の場所に配置します。
2. `index.html` をブラウザで直接開くとページを確認できます。

> **Tip:** Windows PowerShell から開く場合は、プロジェクトルートで次を実行してください。
>
> ```powershell
> Start-Process .\index.html
> ```
